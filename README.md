# ble-lib-template

WebBluetoothを使ったBLE Peripheral制御jsライブラリのテンプレートと、ViteベースのSPAサンプルWebAppを含むモノレポです。

- `lib`: ライブラリ本体（TypeScript, Vite, dist出力）
- `example`: ライブラリ利用サンプルSPA（Vite, TypeScript）

## 開発環境
- TypeScript
- Vite
- ESLint
- Prettier
- GitHub Actions
- npm workspaces

## 公開
- distフォルダをGitHub Pagesで公開
